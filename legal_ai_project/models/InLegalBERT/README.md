# InLegalBERT — Local Model Files

Download all files from:
https://huggingface.co/law-ai/InLegalBERT/tree/main

Place them here:
- config.json
- tokenizer_config.json
- vocab.txt
- special_tokens_map.json
- pytorch_model.bin  (or model.safetensors)

Once these files are present, `build_ml` will load from this folder
instead of downloading from HuggingFace.
